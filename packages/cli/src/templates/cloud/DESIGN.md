# Design

Style guide for the Todo POC wireframes and future app implementation.

## Wireframe primitives

Use Storyboard wireframe components only for spec previews: structural borders, semantic HTML, `disabled` and `danger` states. No brand colors, shadows, or custom typography systems in wireframes.

## App styling (P4)

When the TanStack Start app is scaffolded, shell chrome may use shadcn/Tailwind. Product UI should follow this document as it grows.
